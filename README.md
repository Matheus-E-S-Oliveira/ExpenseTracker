# 💰 Sistema de Controle de Gastos Residenciais

Sistema desenvolvido para auxiliar no gerenciamento de despesas domésticas, permitindo registrar, visualizar e acompanhar gastos ao longo do tempo.

---

# 📌 Objetivo

Permitir que usuários registrem e acompanhem seus gastos residenciais, facilitando o controle financeiro e a organização das despesas mensais.

---

# ⚙️ Tecnologias Utilizadas

## Backend

* .NET
* Entity Framework Core
* MySQL
* MediatR
* Scalar

## Arquitetura

* Clean Architecture
* Mediator Pattern
* Repository Pattern

# 🚀 Funcionalidades

* Cadastro de gastos
* Edição de gastos
* Exclusão de gastos
